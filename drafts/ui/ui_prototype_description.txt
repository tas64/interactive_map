    Клиентский интерфейс представляет собой страницу в верхней половине которой расположена карта местности,
на которой будут отображаться в некотором виде неподвижные объекты, а также маршруты подвижных объектов.
    Нижняя половина страницы состоит из двух списков, левый содержит набор неподвижных объектов с возможностью выбора
для отрисовки на карте, а правый список - набор подвижных объектов, с возможностью выбора для отрисовки маршрутов на карте.
    В центре экрана располож виджет диапазонный слайдер, который позволяет отсекать некоторые точки для частичной отрисовки
маршрута.
